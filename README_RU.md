# Paper Reader (papertok-reader)

Этот репозиторий — кастомный дистрибутив на базе **[Anx Reader](https://github.com/Anxcye/anx-reader)** с интеграцией **PaperTok** и улучшениями **AI/перевода**.

## Статус платформ (важно)

- ✅ Проверено: iOS (iPhone) + iPadOS (iPad)
- ⚠️ Ещё не тестировалось: Android / desktop (macOS/Windows/Linux)

Если вам нужна стабильная мультиплатформенность прямо сейчас, рекомендуется использовать upstream **Anx Reader**.

Документация и инструкции: **[README.md](./README.md)**
